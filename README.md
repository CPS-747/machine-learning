Drawing on a background in Electrical Engineering and the semiconductor industry, I approach machine learning with a focus on robust data pipelines, scalable engineering, and clear, actionable insights.

* **Machine Learning & Modeling:** Scikit-Learn, Keras, TensorFlow, CatBoost
* **Interactive Visualization:** Plotly
* **Environment:** Jupyter Notebooks
